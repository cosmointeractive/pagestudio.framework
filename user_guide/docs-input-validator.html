PageStudio is an open source lightweight Content Management Syste (CMS)

Introduction

Page Studio ships with a simple, convenient facility for validating data and retrieving validation error messages via the Validate class.

Basic Usage

Basic Validation Example

Example

<?php 
    $validate = new Validate();
    $validation = $validate->check($_POST, [
        'username' => [
            'required' => false,
            'min' => 5,
            'max' => 20,
            'unique' => 'users'
        ]
    ]);
?>

Future Syntax

Example

<?php 
    $validation = Validate::check($_POST, [
        'username' => [
            'required' => false,
            'min' => 5,
            'max' => 20,
            'unique' => 'users'
        ]
    ]);
?>
