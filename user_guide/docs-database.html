Qurey Builder Documentation

PageStudio is an open source lightweight Content Management Syste (CMS)

Introduction

The database query builder provides a convenient, fluent interface to creating and running database queries. It can be used to perform most database operations in your application, and works on all supported database systems.

Note: The Page Studio query builder uses PDO parameter binding throughout to protect your application against SQL injection attacks. There is no need to clean strings being passed as bindings.

Selects

Retrieving All Rows From A Table

Example

<?php 
    $posts = DB::table('posts')->get(); 
?>

Retrieving All Rows From A Table: Where Clause

Example

<?php 
    $posts = DB::table('posts')->where('post_status', 'draft')->get();
?>

Retrieving A Single Row From A Table

Example

<?php
    $post = DB::table('posts')->where('author', 'John')->first();
?>

Using Where Operators

Only the following operators are acceptable =, <, >, >=, =<

Example

<?php
    $post = DB::table('posts')->where('post_date', '>=', '2015-05-20')->get();
?>

Inserts

Inserting Single Records Into A Table

Example

<?php 
    DB::table('posts')->insert([
        'title' => 'Hello World!', 'status' => 'draft'
    ]);
?>

Inserting Multiple Records Into A Table

Example

<?php 
    DB::table('posts')->insert([
        ['post_title' => 'Hello World A!', 'post_status' => 'draft'],
        ['post_title' => 'Hello World B!', 'post_status' => 'draft']
    ]);
?>

Note: The [ ] tags in PHP are an alternate way of writing array(). They are an acceptable syntax for the insert() methods.

Updates

Updating Records In A Table

Example

<?php 
    DB::table('posts')->where('id', 1)->update(['status' => 'published']);
?>

Deletes

Deleting Records In A Table

Only the following operators are acceptable =, <, >

Example

<?php 
    DB::table('posts')->where('id', '<', 36)->delete();
?>

Deleting All Records From A Table

Example

<?php 
    DB::table('users')->delete();
?>
